기술

HTML

하이퍼 텍스트 마크업 언어(영어: Hyper Text Markup Language, HTML, 문화어: 초본문표식달기언어, 하이퍼본문표식달기언어)는 웹 브라우저에 표시되도록 설계된 표준 마크업 언어다.

CSS

종속형 시트 또는 캐스케이딩 스타일 시트(영어: Cascading Style Sheets)는 마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어(영어: style sheet language or style language)로[1], HTML과 XHTML에 주로 쓰이며, XML에서도 사용할 수 있다.

JavaScript

자바스크립트(영어: JavaScript)는 객체 기반의 스크립트 프로그래밍 언어이다. 이 언어는 웹 브라우저 내에서 주로 사용되며, 다른 응용 프로그램의 내장 객체에도 접근할 수 있는 기능을 가지고 있다.
